# Laverro

Mobile car detailing in Dubai. Static site with a scroll-scrubbed video background.

## Stack
- Vanilla HTML / CSS / JS — no build step.
- Hero video is a 720p H.264 with frequent keyframes for smooth scroll-scrubbing.
- Hosted on Vercel.

## Local preview
```bash
python -m http.server 8765
# open http://localhost:8765
```
